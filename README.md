## Introduction
<br/>

> *App is way too big to be covered with single pair of eyes.*

<br/>

That's why we are going to automate application testing in order to save time and nerves.

## Installation

1. Install [yarn](https://yarnpkg.com/) by running: `npm i yarn --global`
1. Run `yarn` to install all dependencies
1. Run `yarn dev` to start development version of cypress as web browser to preview tests and execution
1. Run `yarn test` to run project integration test in headless mode with all necessary recordings and screenshots
